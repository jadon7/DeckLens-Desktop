DeckLens

把图片化 PPT、AI 生成页面和截图，还原成可继续编辑的 PowerPoint。保留原版观感，同时恢复文本框和可整理的页面元素。

Capabilities

面向后续编辑，而不是只导出一张图片。

DeckLens 的目标是把最值得继续修改的部分恢复出来：文本可改，关键元素可整理，复杂背景保持稳定。

文字恢复为 PPTX 文本框

识别页面文字、位置和样式，把需要改动的内容恢复成 PowerPoint 里可编辑的文本对象。

底图清理保持原版观感

使用本地清理算法去掉旧文字区域，尽量保持背景连续，避免把复杂页面拆得过碎。

元素分层可人工校对

图文分离后进入预览面板，可选择、合并或删除图层，再生成最终 PPTX。

Workflow

一次导入，四步完成。

桌面端把上传、识别、重建、导出放进一个紧凑流程，适合反复处理多页素材。

1导入

选择 PNG、JPG 或把 PDF 页面按图片导入。

2读取

识别文本、位置、页面结构和基础样式。

3重建

清理底图，并按模式恢复文本或页面元素。

4导出

生成可继续编辑的 PPTX，多页素材合并成一套文件。

Modes

按页面复杂度选择恢复深度。

标准还原

适合只需要快速改字的页面。优先恢复文字，背景保持为清理后的稳定底图。

Fast default
元素分层

适合需要继续改版的页面。先拆出主要元素，再在预览里合并、删除或整理。

Reviewable
AI 智能分层

适合结构更复杂的视觉页面。可配置自己的 fal.ai API Key 接入 Qwen-Image-Layered。
