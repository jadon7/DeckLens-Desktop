# Release 0.2.10

## Context

Recent Agent conversion validation showed that generated icon replacements can
be stretched when the detected source region is not square. This creates visibly
incorrect icons even when the right icon symbol is selected.

## Decision

Release `0.2.10` updates the bundled DeckLens Agent skill to require icon
replacement assets to preserve a 1:1 aspect ratio. Non-square source regions
must center a square icon inside the region, while backgrounds such as circles,
rounded rectangles, or pills are rebuilt as separate shapes.
