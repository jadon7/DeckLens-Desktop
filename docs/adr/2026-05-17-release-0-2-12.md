# Release 0.2.12

## Context

The Windows first-run runtime setup page exposed raw installation logs and
fallback download controls. That is not a product-grade onboarding experience.
Windows users also reported that the generated PPT list and settings entry
buttons were not visible.

## Decision

Release `0.2.12` changes the first-run setup into a simple progress screen:

- Hide raw dependency installation logs from normal users.
- Remove the Python download button and the manual "start DeckLens" button.
- Show a stage-based progress bar and concise status message.
- Continue to enter DeckLens automatically after runtime installation.
- Keep the top-right list/settings controls visible on all Electron platforms,
  not only macOS.

## Package Size Note

Bundling the Python runtime and current heavy image dependencies in the app
package would likely add hundreds of megabytes to more than a gigabyte per
platform, before optional model caches. Keeping runtime setup separate remains
the smaller default installer strategy.
