# Release 0.2.5

## Context

Version 0.2.4 could fail on packaged startup because the Electron main process loaded the Agent Skill runtime from a source-tree path that is not guaranteed to exist inside `app.asar`.

## Decision

- Load the Agent Skill runtime from packaged `Resources/cli/agent-skills.cjs` first, with a source-tree fallback for development.
- Merge the update checker UI into the version card in Settings.
- Publish 0.2.5 as a patch release through the Cloudflare update feed.

## Verification

- Fast packaged build starts without the missing module crash.
- The update feed must point users to 0.2.5 so broken 0.2.4 installs can update.
