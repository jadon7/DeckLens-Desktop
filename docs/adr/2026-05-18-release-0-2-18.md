# Release 0.2.18

## Context

The DeckLens Agent Skill overcorrected toward broad SVG/vector reconstruction. In testing, agents drew too many non-basic objects, missed icon-library replacements, and sometimes changed card geometry.

## Decision

- Constrain vector/shape reconstruction to rectangles, circles, card/button containers, icons, dividers, bar charts, column charts, and simple backgrounds.
- Require flat icons to use the DeckLens icon library instead of emoji, text, or hand-drawn substitutes.
- Preserve arrows, hand-drawn marks, complex decorations, screenshots, logos, 3D, photos, and non-bar charts as original bitmap structure unless they are matched icon-library results.
- Require rebuilt cards and rectangles to match original size, radius, shadow, border, fill, opacity, and gradient.
- Publish 0.2.18 so the packaged app and managed Agent Skill install the updated rules.
