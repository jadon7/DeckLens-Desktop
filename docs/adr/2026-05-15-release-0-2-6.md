# Release 0.2.6

## Context

The Agent Skill needed clearer guidance for element-layer post-processing. Segmented layers can include fuzzy edges and mask artifacts, which can cause agents to treat originally simple graphics as complex raster elements and skip SVG or symbol replacement.

## Decision

- Update the DeckLens Agent Skill to evaluate vector candidates from the original visual region before relying on the extracted layer mask.
- Expand vector candidates beyond rectangles and circles to include arrows, common shapes, badges, dividers, and simple icons.
- Prefer standard icon replacement, such as SF Symbols or Material Design icons, before custom SVG drawing when there is a reliable match.
- Keep unsuitable elements, such as 3D renders, photos, gradients, heavy shadows, textures, screenshots, and detailed logos, as raster layers.
- Publish 0.2.6 so the packaged skill and global skill updater can distribute the guidance.
