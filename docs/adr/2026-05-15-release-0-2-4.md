# 2026-05-15 Release 0.2.4

## Context

DeckLens needs a new public desktop release after adding Agent skill
installation, CLI-based skill installation, and version-managed Agent skill
updates.

## Decision

- Bump the desktop package version to `0.2.4`.
- Bump the bundled `decklens-convert` skill metadata to `0.2.4` so installed
  global skill copies can detect the new source version.
- Publish release artifacts to the Cloudflare-backed update feed after local
  release builds complete.
