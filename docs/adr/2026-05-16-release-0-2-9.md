# Release 0.2.9

## Context

The DeckLens Agent skill now handles vector/icon post-processing well enough for release, but recent validation showed one recurring gap: obvious gradients could be flattened into solid color areas during manual PPTX reconstruction.

## Decision

Release `0.2.9` updates the bundled Agent skill to make gradient/background restoration an explicit conversion responsibility. The skill now requires Agents to identify gradient targets before segmentation, preserve or rebuild them during post-processing, and report how those gradients were restored.
