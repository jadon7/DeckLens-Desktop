# Release 0.2.11

## Context

Windows first-run setup could stop with a raw Python-not-found error, leaving
users without a clear path to install the required Python 3.12 runtime. The
setup screen also used a dark visual treatment that no longer matched the main
DeckLens workbench.

## Decision

Release `0.2.11` updates the setup experience:

- Detect common user and system Python 3.11/3.12 install paths on Windows.
- When Python is missing on Windows, try installing Python 3.12 through
  `winget` before failing.
- Provide a direct Python 3.12 download action for machines without `winget`.
- Restyle the setup screen to match the main product UI: white canvas,
  restrained borders, centered headline, pill buttons, and a clean log panel.
